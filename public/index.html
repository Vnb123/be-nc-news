Introduction

NC News API is designed to be an interactive news interface where users can search for and read articles, add and delete comments and vote on them.

  • Topics
  • Articles
  • Users

Topics

GET /api/topics

This will be an object representing the topics.

topics": [{ "slug": "football", "description": "Footie!"}]
